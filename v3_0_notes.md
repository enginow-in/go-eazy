# Release Notes - v3.0

Welcome to **Go-Eazy v3.0**! This major release brings fundamental architectural improvements, robust access controls, and a cleaner codebase to set the stage for future growth.

## 🚀 Highlights & Features

- **Robust Role-Based Access Control (RBAC)**
  - Implemented advanced RBAC via Supabase specifically designed for the `SystemAdmin` role.
  - Ensures secure, scalable, and manageable access management across the platform.

## 🛠 Improvements & Refactoring

- **Enhanced Nearby Services Filtering**
  - Updated the filtering system to seamlessly and cleanly support an 'All Cities' view.
  - Improved data fetching efficiency for regional queries.
- **Core Architecture Cleanup**
  - Resolved map removal dependencies.
  - Cleaned up core project files to reduce technical debt and improve maintainability.
